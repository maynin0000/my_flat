# MVP PRD

## 목표
- 사용자가 “지금 사도 되는지”를 가격 추적으로 판단
- “특정 사이즈 재입고”를 관측 기반으로 빠르게 알림

## 핵심 기능(P0)
1. 상품 링크 등록
2. 가격 추적(표시가 기준)
3. 사이즈 품절 여부 추적
4. 특정 사이즈 재입고 알림
5. 쿠폰 입력 기반 예상가 표시

## 정책(B안)
- 관측 기반 서비스
- UI에 마지막 확인 시간 노출
- 확인 요청 알림으로 관측 공백 보완

## KPI(MVP)
- 트래킹 등록 수
- 재입고 알림 클릭률
- 알림→상품 화면 진입률
- 상품 화면→외부 구매 링크 클릭률
